Intelligent Agricultural System (农业智能视觉问答系统)
📖 项目简介 (Project Overview)
本项目是一个基于多模态知识图谱与大语言模型 (LLM) 的农业智能辅助诊断与问答系统。
针对传统农业生产中“农户难以准确描述病虫害症状”、“检索门槛高”以及“通用大模型易产生事实幻觉”三大痛点，本系统创新性地打通了**“视觉感知 -> 图谱溯源 -> 语义表达”**的全链路闭环。

系统允许农户通过上传农作物图片进行病虫害智能识别，并在底层依靠严谨的 Neo4j 农业知识图谱进行事实锚定，最终由 LLM 转化为通俗易懂的农技处方，实现如农业专家般“望、闻、问、切”的智能化交互体验。

✨ 核心功能模块 (Core Features)
👁️ 图片识别与检索 (视觉感知)：基于 YOLO 目标检测算法，支持对上传的农作物图片进行实时的病虫害特征提取与定位。

🧠 AI 智能问答 (大模型 RAG)：结合 LLM 与图谱检索增强生成，消除大模型在农业垂直领域的“幻觉”，提供 100% 严谨的自然语言农技指导。

🕸️ 知识图谱可视化 (实体与关系查询)：将非结构化农业数据结构化，支持对作物类别、病害属性、农药关系进行图形化的层级下钻与拓扑查询。

⚡ 多维度农知概览：底层结合 KNN 算法对海量离散农业数据进行自动化的分类与标签预测，实现标准化知识的敏捷展示。

🛠️ 技术架构体系 (Tech Stack)
本项目采用前后端分离与微服务架构思想，集成异构算法矩阵：

前端交互层：响应式 Web 界面 (支持图谱可视化渲染)

后端服务层：Django (核心业务调度与 API 路由)

知识构建层 (KG)：

Scrapy：海量农业语料爬虫

Keras (深度学习)：农业文本命名实体识别 (NER) 与关系抽取

KNN (机器学习)：结构化数据的自动化标签预测 (predict label)

多模态与认知层：

Ultralytics (YOLO)：端到端图像目标检测

LLM + Prompt Engineering：自然语言理解与推理决策

数据存储层：Neo4j (图数据库)
